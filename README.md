# Rock Paper Scissors Game
An interactive, animated, and Responsive Rock Paper Scissors game built using HTML, CSS, and JavaScript — featuring a sleek dark theme and real-time score tracking!

🔗 Live Demo  
https://utsavthakur448.github.io/Rock-Paper-Scissor-Game/  


## 📚 How to Use
1. Clone the repository:
   ```bash
   git clone https://github.com/utsavthakur448/Simon-Says-Game.git
2. Open index.html in your browser.  


🧠 How It Works
- You pick: ✊ Rock, 🖐️ Paper, or ✌️ Scissors
- The computer randomly picks its move
- Winner is declared instantly on screen 🏆
- Scores are tracked live for both you and the computer  

✨ Features  
✅ Dark theme for modern look  
✅ Responsive layout for mobile and desktop 📱💻  
✅ Real-time score updates  
✅ Smooth hover and scale animations  
✅ Displays computer's choice after every round  
✅ No reloads — pure JavaScript interactivity  
  

🛠️ Built With
- HTML
- CSS
- JavaScript
